# Changelog

All notable changes to the PAYUNi plugin will be documented in this file.

## [1.0.0] - 2025-01-27

### Added
- Initial release
- `payuni` - Overview and environment setup skill
- `payuni-checkout` - UPP checkout integration skill
- `payuni-query` - Transaction query skill
- `payuni-webhook` - Webhook handling skill
- Code examples for Node.js/TypeScript and PHP/Laravel
- Error codes reference
- Troubleshooting guide

### Features
- AES-256-CBC encryption/decryption
- SHA256 HashInfo generation
- CheckCode signature verification with timing-safe comparison
- Multi-framework support (Next.js, Express, Laravel)
- Replay attack prevention guidance
