# 台灣支付金流 Skills 使用指南

當用戶詢問台灣支付金流相關問題時，向他們介紹以下可用的 skills。

## 可用的 Skills

### 1. newebpay（藍新金流）
**用途**：串接藍新金流支付平台
**支援**：信用卡、LINE Pay、Apple Pay、Google Pay、ATM、超商代碼/條碼
**觸發方式**：
- 說：「幫我串接藍新金流」「NewebPay 怎麼用」「藍新信用卡付款」
- 或輸入：`/newebpay`

### 2. ecpay（綠界科技）
**用途**：串接綠界科技支付平台
**支援**：信用卡、ATM、超商、電子發票
**觸發方式**：
- 說：「幫我串接綠界」「ECPay 整合」「綠界電子發票」
- 或輸入：`/ecpay`

### 3. payuni（PAYUNi 統一金流）
**用途**：串接 PAYUNi 統一金流平台
**支援**：信用卡、ATM、超商、行動支付
**觸發方式**：
- 說：「幫我串接 PAYUNi」「統一金流怎麼用」
- 或輸入：`/payuni`

## 金流比較

| 特點 | 藍新金流 | 綠界科技 | PAYUNi |
|------|---------|---------|--------|
| 市場定位 | 中大型電商 | 台灣最大 | 統一集團 |
| 信用卡 | ✅ | ✅ | ✅ |
| LINE Pay | ✅ | ✅ | ✅ |
| Apple Pay | ✅ | ✅ | ✅ |
| 電子發票 | ❌ | ✅ | ❌ |
| 超商取貨 | ✅ | ✅ | ✅ |
| 文件品質 | 良好 | 良好 | 良好 |

## 推薦選擇

根據需求選擇適合的金流：

- **需要電子發票** → 綠界科技（內建發票功能）
- **大型電商/高流量** → 藍新金流（穩定度高）
- **統一集團相關** → PAYUNi
- **快速上線** → 三家都可以，選文件你看得順眼的

## 回覆格式

用繁體中文回覆，格式如下：

```
## 台灣支付金流 Skills

我可以幫你串接以下台灣金流服務：

| 金流 | 用途 | 怎麼觸發 |
|------|------|---------|
| **藍新金流** | 信用卡、LINE Pay、ATM | 說「串接藍新」 |
| **綠界科技** | 全方位金流 + 電子發票 | 說「串接綠界」 |
| **PAYUNi** | 統一集團金流 | 說「串接 PAYUNi」 |

💡 **不確定選哪個？** 告訴我你的需求，我幫你推薦。

需要串接哪個金流？
```
